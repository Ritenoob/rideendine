# RideNDine Project

## Overall Project Context
The RideNDine project is an innovative solution aimed at enhancing ride services through a seamless integration of technology and transportation. The project's goal is to provide comprehensive and flexible services for users and administrators alike, ensuring an optimal experience.

## MCP Server Setup
To set up the MCP (Modular Communication Protocol) server, follow these steps:
1. **Install MCP Server**: Ensure that the MCP server is installed on your system.
2. **Configuration File**: Edit the `mcp_config.yaml` file located in the installation directory. Specify the server host, port, and other necessary parameters.
3. **Start Server**: Use the command `mcp_server start` to initialize the server.

## Logging Configuration
For effective logging, configure the `logging.yaml` with the following settings:
- **Level**: Set the logging level to `INFO` for general operational logs and `DEBUG` for detailed tracking.
- **Output Format**: Use a structured format (e.g., JSON) to facilitate easier parsing and analysis.
- **Log Rotation**: Enable log rotation to manage disk space, specifying the maximum file size and number of backup files.
