# API Service (NestJS)

**Status:** In progress (not wired to other services)

## Purpose
Core REST API for auth, users, orders, and payments in the planned service split.

## Run (dev)
From `services/api`:

```
npm install
npm run start:dev
```

## Demo Stub
There is a demo health stub at `services/api/server.js` used by the core demo tooling.

## Notes
- This service is not yet integrated with dispatch/routing/realtime.
- See [services/README.md](services/README.md) for context.
