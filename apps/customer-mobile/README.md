# Customer Mobile (Expo)

**Status:** Prototype (demo tracking client)

## Run
From `apps/customer-mobile`:

```
npm install
npx expo start
```

## Usage
- Server URL: `http://<your-computer-ip>:8081`
- Order ID: copy from the dispatch demo dropdown
- Deep link example: `ridendine://track?orderId=YOUR_ORDER_ID`

## Notes
- Requires the core demo server to be running.
- Uses React Native Maps for live tracking.
